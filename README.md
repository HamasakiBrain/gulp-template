# Front-end-template

## Quickstart

### 1. Install
Клонируем репозиторий
```shell
git clone git@github.com:Maizerer/gulp-template.git
```
Устанавливаем gulp глобально
```shell
npm install gulp -g
```
Устанавливаем зависимости
```shell
npm install
```
### 2. Start server

```shell
gulp default
```
